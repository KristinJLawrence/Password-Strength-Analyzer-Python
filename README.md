# Password Strength Checker

## Description
This project is a Python-based password strength checker.  
It evaluates whether a password meets basic security requirements.

## Features
- Minimum length check (8 characters)
- Requires uppercase and lowercase letters
- Requires at least one number
- Requires at least one special symbol
- Provides clear feedback to the user

## Why This Project Matters
Weak passwords are one of the most common security risks.
This script demonstrates how password policies can be enforced programmatically.

- Python 3
- Command Line (Windows)
